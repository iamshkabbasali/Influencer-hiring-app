📱 Influencer Hiring Application

The Influencer Hiring Application is a full-stack mobile platform that connects brands with social media influencers for marketing collaborations. It allows brands to create campaigns and hire suitable influencers, while influencers can discover opportunities, showcase their work, and communicate with brands in a seamless and organized way.

🚀 Features

🔐 User Authentication (Login / Register with JWT)

👥 Role-Based System (Brand & Influencer)

📢 Campaign Management (Create, View, Apply)

📩 Application Handling (Accept / Reject Influencers)

💬 Real-time-like Messaging (Text & Image)

🖼 Portfolio Upload System

📸 Social Features (Posts, Likes, Comments)

👤 Profile Management

➕ Follow System (like social media)

🔔 In App Notifications System

🔎 Search Users

⚙️ How It Works

Brands create campaigns and manage influencer applications.

Influencers browse campaigns and apply based on their interests.

Both users can communicate through the messaging system.

Influencers can upload portfolios and posts to showcase their work.

📌 Future Improvements

Real-time chat using Socket.io

Advanced analytics for influencers

Push notifications

📖 Purpose

This project demonstrates full-stack mobile app development using Flutter, Node.js, and MySQL. It showcases concepts like REST APIs, authentication, database design, and role-based access control, making it a strong academic and portfolio project.
